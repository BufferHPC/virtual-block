####项目启动方式

```js
//通过当前gitHub地址下载源码后，运行下面命令，安装相关依赖包
npm install
```

```js
//开发环境运行，自动编译和热重载
npm run serve
```

```js
//生产环境运行，自动编译和压缩后输出
npm run build
```

更多自定义配置请前往官方文档： [配置参考  Configuration Reference](https://cli.vuejs.org/config/).
